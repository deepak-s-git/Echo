# Contributing to Echo

Thank you for your interest in contributing to Echo! We are all about good vibes and building the best workflow memory utility for macOS. 

Here are a few guidelines to help you get started:

## How to Contribute

### 1. Reporting Bugs
- Search existing issues before creating a new one.
- Use a clear and descriptive title.
- Provide steps to reproduce the bug, along with your macOS and Xcode versions.

### 2. Suggesting Enhancements
- Open an issue describing the proposed feature.
- Explain why this enhancement would be useful to most users.

### 3. Pull Requests
- Fork the repository and create your branch from `main`.
- Ensure your code builds cleanly with no warnings or errors.
- Maintain consistent code formatting and coding conventions (Swift/SwiftUI).
- Write descriptive commit messages.

## Code of Conduct
Please be respectful and constructive in all interactions. Let's keep this project welcoming for everyone!
